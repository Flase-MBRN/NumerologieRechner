## NACH DEM GENERIEREN

### Schritt 1: Bild auswählen
- Wähle das Bild das am besten passt
- Klicke auf "Download"

### Schritt 2: Größe anpassen
Bing erstellt meist 1024x1024. Du musst es auf **1200x630** zuschneiden:

**Option A: Canva (Kostenlos)**
1. https://www.canva.com/
2. "Custom Size" → 1200 x 630 px
3. Bild hochladen
4. Als Hintergrund setzen
5. Download als PNG

**Option B: Photopea (Kostenlos, wie Photoshop)**
1. https://www.photopea.com/
2. File → New → 1200 x 630
3. Bild reinziehen
4. File → Export → PNG

**Option C: IrfanView (Windows)**
1. Bild öffnen
2. Image → Resize/Resample
3. Canvas Size → 1200 x 630
4. Save as PNG

### Schritt 3: Dateiname
Speichern als: **`og-image.png`**

### Schritt 4: Speicherort
Legen Sie es ins **Root-Verzeichnis** (neben index.html)
